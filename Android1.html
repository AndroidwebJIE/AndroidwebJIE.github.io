Historia de Android Características Usos y compatibilidad

Usos y compatibilidad

Android es el sistema operativo más popular en la telefonía móvil, siendo utilizado como principal SO de la mayoría de compañías principales, tales como Samsung, Xiaomi…

Dentro de las distintas versiones que ha ido desarrollando Android (+info) la compatibilidad de este sistema operativo va aumentando con cada una de ellas.

SDK Android

SDK (Software Developer Kit) es un conjunto de herramientas de desarrollo de software que permite a un desarrollador la creación de una aplicación informática enfocada a un sistema en concreto.

Las más comunes permiten el uso de específicos lenguajes de programación o incluso introducir hardware específico para una comunicación precisa con su sistema de comunicación.

Un SDK es generalmente texto (ya sea por los programas, instrucciones o licencias), salvo por algunas excepciones que incluyen hardware.


En concreto, el SDK de Android (elaborado por Google gracias a sus similitudes) incluye un depurador de código, biblioteca, un simulador de teléfono basado en QEUM, documentación relevante, ejemplos de código y tutoriales.

Su plataforma integral de desarrollo o IDE (Integral Development Enviroment) soportada oficialmentes es Android Studio que incluyen soportes para la detección de errores, anexo al complemento ADT (Android Development Tools Plugin) que es un plugin para Eclipse que permite la construcción de las apps para Android.

Una de las ventajas con las que cuentan los programadores, es que pueden utilizar un editor de texto para utilizar ficheros Java y XML y usar comandos en un terminal concreto (con la ayuda de los paquetes JDK y Apache) para la creación y depuración de las aplicaciones, teniendo la opción de controlar dispositivos Android (conectados) en remoto.


Actualizaciones y Características SDK

Sus actualizaciones están organizadas por el desarrollo general de Android. Aunque las versiones antiguas no desaparecen por si fuera necesario para algún programador que las requiera para la instalación de aplicaciones en dispositivos desfasaos o anticuados. Ya que cualquier herramienta de desarrollo son elementos descargables teniendo la versión que se halle en el dispositivo, probando así versiones anteriores y comprobaciones de compatibilidad.


Android utiliza el ADB (Android Debug Bridge; Conector/puente para depuración Android). Conjunto de herramientas incluido en el SDK de Android que desde línea de comandos se encarga de comunicar mediante programas con función tanto de cliente como de servidor.

Fastboot (arranqe rápido), es un protocolo de diagnóstico cuya función principal reside en modificar el sistema de ficheros flash mediante una conexión USB desde una computadora. Donde podrá arrancarse en modo "Boot Loader" o "Second Program Loader". Tras habilitar este protocolo en el dispositivo, se acepta un cúmulo de intstrucciones transmitidas mediante la conexión USB y el uso de línea de comandos.

Android Studio

Es el entorno de desarrollo integrado oficial para la plataforma Android. Publicando su primera versión estable en diciembre de 2014. Disponible para Windows tanto en plataformas de 32 como de 64 bits, para GNU/Linux con 2GB de memoria RAM mínimo y para macOS.

Basado en el software IntelliJ IDEA (Entorno de Desarrollo Integrado para el desarrollo de programas informáticos y desarrollado por JetBrains), Android Studio admite los mismos lenguajes de programación de este software, como Java y C++ entre otros. En versiones posteriores a Android 3.0 también es compatible Kollin (lenguaje preferido de google).

Este IDE consta de herramientas Lint (utilizado para detectar problemas de rendimiento, compatibilidad de versiones, ect.), funciones de firma de aplicaciones, plantillas con las que poder crear diseños comunes de Android junto con un editor que permite la visualización de cada componente de la interfaz de usuario únicamente arrastrando y soltando. También consta de un soporte para programar aplicaciones para Android Wear (Dispositivos corporales basados en Android) o Google Cloud PLatform. Y dispone de un dispositivo virtual de Android que se utiliza para la ejecución y prueba de aplicaciones, de poder hacer renderizados a tiempo real, y de una consola de desarrollador (donde se pueden encontrar consejos de optimización, ayuda y estadísticas de uno).

Dispositivos que utilizan Android

Los más comunes

Teléfono inteligente

El dispositivo más conocido y utilizado, Android apareció para los móviles. Estos funcionan muy bien con este sistema operativo. Generalmente de un tamaño pequeño, fácil de manejar suele disponer de conectar a internet y memoria suficiente para almacenar archivos, también diseñado para funciones generales.

El primer teléfono móvil disponible para ejecutar Android fue el HTC Dream.


Tabletas

Las tabletas no son tan funcionales como los smartphones con Android, pero son dispositivos bastante estudiados por Android y se podría llegar a decir que han habido versiones específicas para estos. De tamaño un poco mayor que los móviles, también de pantalla táctil son un elemento al que se le puede sacar mucho partido, tanto para el aprendizaje como para el ocio. Con acceso a múltiples aplicaciones, estos dispositivos se pueden utilizar en cualquier parte y realizar tareas rutinarias como revisar el correo.


Smartwatch o smartband

Algunas compatibles con Google y por tanto con google play, también tienen el privilegio de formar parte de los dispositivos utilizados y dirigidos mediante Android. Apareciendo así "Android Whear", el cual ha sido muy valorado y criticado por diferentes posturas.

Sí que es cierto que no es muy común, pero poco a poco se van haciendo con el mercado. Conectándose a nuestros móviles, desempeñan varias funciones, tales como un seguimiento de nuestro estado físico, acceso GPS, domótica...


Ordenadores

Además de poder ejecutar aplicaciones y juegos, Android PC (encontrado tanto en ordenadores de sobremesa como en portátiles) da la posibilidad de conectar con la cuenta de Google Play y hacer así uso de todas nuestras apps y de todos sus servicios, convirtiendo una plataforma móvil en una que puede tener sentido en el escritorio. Han existido actualizaciones aunque ninguna ha llegado a ser notable.


Accesibilidad en el coches

Android Auto Es el asistente en carretera que facilitar tu viaje, únicamente mediante la voz o tu dedo podrás conducir sin distracciones. Este te permite realizar cualquier acción de tu teléfono sin tener que abrirlo. Conectando tu móvil antes de salir a la carretera mediante USB o de manera inalámbrica, tendrás acceso a tus aplicaciones, tales como google Maps.


Televisores

Android TV llegó prácticamente al mismo tiempo que Android Wear, con el objetivo de poder disfrutar de Android en una pantalla de 55 pulgadas. Permite conexsión Chromecast dando la opción de enviar rápidamente al televisor archivos multimedia (videos, música, fotos...) desde tu dispositivo móvil. Presentado el 25 de junio de 2014 por Google TV. Este proyecto fue desarrollado por Google, Intel, Sony y Logitech.


Los menos comunes

Consolas

Fabricantes como Nvidia o Razer se fijaron en el avance que pudo llegar a ser Android TV y comenzaron a desarrollar lo que sería la primera consola Android, donde se pueden ejecutar todos los títulos disponibles en Google Play y tener la opción a descargar emuladores con los cuales poder procesar juegos. Aunque la calidad de los procesos y experiencia no sea excelente, se ha conseguido el objetivo principal de poder procesarlos, pero sí que se han conseguido una superior ejecución los juegos más populares de Google Play.


Espejos

Una gran novedad ha sido la aparición de "espejos inteligentes", donde presentan una pantalla de información adicional mientras nos miramos en él. Teniendo además de nustro reflejo, datos metereológicos, noticias, información del tráfico actualizada, comprobación de nuestros mensajes, correos o demás notificaciones y algunos aspectos del estilo. Su estructura es a priori sencilla, se ha empleado para conseguirlo un espejo de doble cara, teniendo la pantalla entre dos caras de espejo junto con una placa conectada a un Fire TV y algunas líneas de código para tener la opción de widgets y APIs de Android. Teniendo todos los componentes escondidos en el mueble situado en la parte trasera del espejo proporciona una elegancia y sensación de "mágico" que nos permite disfrutar plenamente de sus funciones.


Proyector

Su característica básica es la comodidad, ya que al tener su propio sistema operativo, nos podemos ovidar de tener la obligación de conectarlo a otro dispositivo. Con unas resoluciones de calidad, acceso a Wifi y Bluetooth, sonido claro y potente y una pantalla de hasta 200 pulgadas, Android nos da la opción de descubrir un proyector android. Por lo general suelen ser dispositivos grandes/incómosod para el uso doméstico, y dependiendo de la marca tendrán más o menos propiedades extras. Algunos modelos que nos podemos encontrar son OTHA proyector Mini portátil, WiMiUS K5 y Artlii Play Proyector Smart entre otros, son los que dan forma a esta idea tan revolucionaria que cada vez podemos encontrar más en el mercado.


Nevera

Puede parecer una idea alocada, pero el concepto de nevera con Android se puede volver algo completamente cotidiano si lo incluimos en nuestro día a día, ya que por lo general nuestras neveras pueden ser un buen lugar de "encuentro" en el que dejar anotaciones, calendario, fotografías o detalles que pueden dejar de ser algo físico. La nevera Android, presentada por primera vez en 2016, además de dar la opción a redactar una lista de la compra conectada con tu própio dispositivo móvil, permite la reproducción de archivos multimedia mediante el uso de sus aplicaciones, puediendo así cocinar con música, serie o cualquier tipo de expresión digital. Los nuevos electrodomésticos llegan a nuestras vidas de una manera impactante, ya que muchas de las acciones que desempeñan hace no mucho podían considerarse de ciencia ficción. La nevera Android Dispone de dos puertas laterales, dividiendo el congelador del frigorífico con una capacidad de hasta 593 litros, dispensador frontal de hielo, una pantalla de 21'6 pulgadas que nos da acceso a multitud de funciones (una d ellas es poder visualizar el interior de la nevera sin necesidad de abrirla).


eReaders

Estos libros electrónicos Android principalmente son conocidos por la merca Energy Sistem. Con 16 niveles de grises y un sistema anti-reflectante, posee ua resolución de 166ppp que proporcionan una satisfactoria experiencia de lectura. Estos dispositivos poseen botones laterales que permiten un fácil manejo, para pasar página, dispone de aplicaciones integradas con las que podrás conectarte a redes sociales con el fin de poder compartir frases o citas de esa lectura o simplemente tener la oportunidad de compartir información de manera más fácil. Este dispositivo, al utilizar la tecnología de tinta electrónica consumen muy poca batería, locual es un punto muy positivo.


TPV

Las pantallas TPV (Terminal punto de venta), son dispositivos utilizados casi por cualquier bar o tienda para almacenar todos los datos, gestionar tareas y transacciones realizadas en el comercio. Android ha entrado de lleno en este mercado.
